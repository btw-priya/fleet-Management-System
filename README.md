# Fleet Management System

A simple Java Console-based CRUD (Create, Read, Update, Delete) application for managing vehicle records. This project demonstrates the basic concepts of Java, Object-Oriented Programming (OOP), and ArrayList.

## Features

- Add a new vehicle
- View all vehicles
- Update vehicle details
- Delete a vehicle
- Menu-driven console interface

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner
- Visual Studio Code

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/btw-priya/fleet-Management-System.git
```

2. Open the project folder.

3. Compile the program:

```bash
javac FleetManagementSystem.java
```

4. Run the program:

```bash
java FleetManagementSystem
```
